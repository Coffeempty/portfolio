jax-cce

Published a Python package that cuts memory usage by 97% for the most expensive step in LLM training. Cross-entropy loss in models like Llama-3 creates a 4GB logits tensor that gets used once and thrown away — jax-cce tiles over vocabulary chunks instead, never materialising the full tensor. Same math, same gradients. Reviewed by a JAX core maintainer at Google DeepMind.
